# 微信文章导出器 - Electron 桌面端

## 1. 产品定位

一款独立的 Windows/Mac 桌面客户端，直接运行在用户本地电脑上，**不依赖任何后端服务**。用户通过微信扫码登录后，可采集目标公众号文章并导出为电子书。

## 2. 核心功能

### 2.1 微信扫码登录
- 显示二维码，用户用微信扫描登录自己的公众号
- 登录成功后保存 Cookie 和 Token 到本地
- 支持自动登录（下次打开自动恢复登录态）
- 纯本地运行，不上传任何数据到服务器

### 2.2 公众号采集
- 输入公众号名称进行搜索
- 显示公众号基本信息（头像、名称、文章数）
- 一键采集该公众号的所有文章
- 实时显示采集进度

### 2.3 电子书导出
- 支持导出格式：EPUB（主要）、PDF、DOCX
- 可自定义书名、作者
- 一键导出到用户选择的目录

## 3. 界面设计

### 3.1 窗口规格

- **窗口尺寸**: 420px × 600px（固定大小，不可调整）
- **窗口样式**: 无边框自定义标题栏，包含最小化、关闭按钮
- **位置**: 屏幕居中显示

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 框架 | Electron 33 + Vue 3 (纯 Vue，不使用 Nuxt) |
| UI | 原生 CSS + Vue 组件（轻量简约） |
| 数据库 | better-sqlite3（本地 SQLite） |
| HTTP | axios（直连微信 API） |
| EPUB | epub-gen 或 JSZip 自定义生成 |
| PDF | jspdf + html2canvas |
| DOCX | docx |
| 打包 | electron-builder |
| 样式 | 纯 CSS（不使用 Tailwind 等框架） |

### 4.3 微信 API 调用策略

由于微信 API 需要登录态，且存在 CORS 限制，**所有微信 API 请求都在 Electron 主进程中发起**，通过 IPC 传递给渲染进程：

1. **渲染进程** → 发起 IPC 请求到主进程
2. **主进程** → 使用 axios 发起微信 API 请求（无 CORS 问题）
3. **主进程** → 返回数据给渲染进程

### 4.4 数据存储

使用 `better-sqlite3` 在本地存储：

```sql
-- 账号表
CREATE TABLE accounts (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  nickname TEXT,
  fakeid TEXT,
  cookie TEXT,
  token TEXT,
  appmsg_token TEXT,
  weixin_2021_1 INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 文章表
CREATE TABLE articles (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  account_id INTEGER,
  fakeid TEXT,
  title TEXT,
  link TEXT UNIQUE,
  content TEXT,
  author TEXT,
  source_url TEXT,
  cover_image TEXT,
  publish_date DATETIME,
  read_count INTEGER DEFAULT 0,
  like_count INTEGER DEFAULT 0,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (account_id) REFERENCES accounts(id)
);

-- 设置表
CREATE TABLE settings (
  key TEXT PRIMARY KEY,
  value TEXT
);
```
