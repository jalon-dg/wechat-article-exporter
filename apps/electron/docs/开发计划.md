## 技术选型说明

1. **纯 Vue 3 + Vite**：不使用 Nuxt，减少复杂性，更适合小型应用
2. **纯 CSS**：不使用 Tailwind 等框架，保持代码轻量
3. **主进程发起 API 请求**：避免 CORS 问题，简化架构
4. **better-sqlite3**：成熟稳定的 SQLite 方案
